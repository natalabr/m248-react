# m248-react

## Installation

1. **Clone the repository**
    ```bash
    git clone https://github.com/natalabr/m248-react.git
    cd ./m248-react/
    ```

2. **Install Dependencies**
s
    ```
    npm install 
    ```

3. **Run the Project**
    ```
    npm run dev
    ```

## Technologies Used

- [Vite](https://vite.dev)
- [React](https://react.dev)
- JavaScript
